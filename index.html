Instructions

All ares of this visualization are designed to be explored by the user. By default, this page shows the total views of the IMDb top 250 films as streamed by 1Channel.ch. Interacting with the titles list will filter the data and update all views. Individual titles can be selected by clicking, multiple by click-drag. More granular selections made by command or control clicks. The scatter plot supports brushing and seletion of indiviudal titles. Both the map and bar chart will highlight individual elements and present more detailed site information

Description

If we are to draw informed conclusions about the issue of online piracy, we need to better understand the darker corners of the web that are contributing to it. This project explores a content-indexing website, 1channel.ch. Among the many sites that provide similar information, 1channel.ch had the highest Alexa rank in the US(297, 167). Users on 1channel.ch can access over 49,000 movies and over 6,000 TV shows, all for “free.” Some of the most popular movies on the site have over 1,000,000 views. We are going to be scraping 1channel.ch for the top 250 films on IMDB, using a Python script. At a very high level, our script will do the following: Retrieve the top 250 films from IMDB. Open 1channel.ch and build a search string for the movie. Open the search results page and grab the appropriate movie data. Retrieve the list of third party links and the number of times they have been viewed. More details about our project, the processm, and design decisions can be found in our process book.
